# TGC Project

Dự án website cho TGC.

## Cách chạy

1. Cài thư viện:
```bash
npm install
```

2. Chạy trên máy (vào http://localhost:5173):
```bash
npm run dev
```

3. Build khi xong:
```bash
npm run build
```

## Lưu ý
- Dùng Tailwind CSS để sửa giao diện.
- Ảnh nằm trong `src/assets`.
- Các trang chính nằm ở `src/pages`.
